# Comandos de manutenção - Manjaro/Arch

#### Lista sdk instalado
````shell
$ dotnet --list-sdks
````

#### Lista runtime instalado
````shell
$ dotnet --list-runtimes
````

#### Instalar sdk
````shell
$ pamac install dotnet-sdk   
````

#### Instalar runtimes
````shell
$ pamac install dotnet-runtime
````
